# 1、🤔什么是皮质醇？

皮质醇使人体的一种对抗外界压力的激素，用于面对生存压力时提升专注力和应激反应功能并降低非必要的功能来保障生存的激素。

举个例子：当猎物面对野兽时，皮质醇会迅速上升来迫使猎物迅速反应来摆脱危险，如提高肌肉能力，心肺能力，提高专注效率等，同时抑制消化系统生殖系统，大脑理智功能活动。

这种激素的存在对于生物的繁衍确实起到了重要的作用，但是原始环境的压力不持久，皮质醇来得快去的快的情况不会对生物产生太多负面反应。但是在现代社会中，个人收受到了社会的无形的持久的压力，就会长期的分泌皮质醇，导致一些的负面问题。

# 2、🥲皮质醇的负面问题

1. 大脑：降低记忆力，容易情绪化
2. 神经：容易疲惫，精神萎靡，抑郁，焦虑
3. 肌肉：肌肉损失加速
4. 睡眠：降低睡眠质量

# 3、🫵如何降低皮质醇？

4. 学习，实践。📖
5. 直视恐惧与焦虑。👀
