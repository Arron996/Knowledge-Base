

# 前言

十一月结束了短视频的烦腻之后，开始重新拾起了阅读，以平复我无聊的心情。读了诸多书籍之后，感到确实应该写一下书评来回顾所看的书籍。本月在某人的推荐下，读了两本书。一本是张爱玲的短篇小说《小艾》，另一本是汪曾祺的《人间草木》。关于这两本书的评价会在下面提及。

# 已读

## 《卡拉马佐夫兄弟》

从大学开始读的，难以想象这本书从大学图书馆看到了工作后的长宁图书馆，最后终结与手机上的阅读软件。篇幅之长堪比《丰乳肥臀》、《百年孤独》。故事描写了卡拉马佐夫一家的故事，卡拉马佐夫一家的父亲嗜赌成性，大哥风流成性，不行基督，万事皆允；二哥是一个是俄国的先进知识分支，向往欧洲的秩序，提倡理性；三弟师从佐西马长老，信耶稣，爱众人；四弟也就是父亲的私生子，拥有最丑恶的灵魂。一家子全集齐了，矛盾不可消停。起因是大哥花了老婆的钱，同时对他爸试图抢走他的心上人而怀恨在心，二哥保持理性不参与，三弟爱着众人，就在一个月黑风高夜，大哥醉酒寻找情妇，猜测情妇已经上他父亲哪儿去了，于是想要弑父取钱，但是并没有发生，反而被四弟偷偷补了刀，罪责嫁祸给大哥，二弟默不作声，三弟仁爱众人。。。结果是大哥判了流放，二弟内心受到谴责最后发疯，三弟仍然仁爱众人，四弟上吊自杀。

这本书好长，上半部分读的很爽，主要以三弟的视角写的。中间的一段印象很深刻，讲的是佐西玛长老年轻

被绿了，他想要与对方决斗，但是决斗前的一晚，他开始忏悔，为什么要伤害一个无辜的人，于是他开始仁爱众人。

“我配让别人侍候我，而且就因为他们的贫穷和愚鲁，就该让任意支使他们么？”我当时很奇怪，为什么这样简单的思想，这么清楚的思想，却这么晚才出现在我的脑筋里。世界上固然不可能没有仆人，但是应该设法使你的仆人在精神方面比他即使不做仆人时还要更为自由些。为什么我不能做我仆人的仆人，并且让他看见，在我的方面并无骄色，在他毫不产生猜疑呢？为什么我的仆人不能就像是我的亲人一样，使我最后可以把他列为我家庭的一员，并且引以为快呢？甚至现在也可以做到这一点，作为将来的、人类伟大团结的基础，在那个时候，人将不再找仆人，不愿再像现在的样子，把同样的人当仆人看待，相反地，将照新约的精神，尽力做大家的仆人。人最终将只在教化和慈爱的功业中寻到他的快乐，而不像现在那样在残忍的欢愉，例如贪食、淫荡、虚饰、夸耀和互相嫉妒、竞争中寻找快乐，难道这只是一个梦想么？我深信绝不是梦想，而且这样的时间就要临近了。有人会嘲笑地问：这样的时间究竟什么时候来到，而且确实像是要来到了吗？我想我们和基督在一起，总会完成这伟大的事业的。在人类的历史中，世界上曾有过多少理想，甚至在十年以前还认为不可思议的，却竟能在时间悄悄来临的时候忽然出现，风行整个大地。我们这里也一定会这样，我们的民众将会赫然显现在世界面前，所有的人们将会说：“一块曾被建筑师嫌弃的石头竟成了基石。”我们倒要反问那些嘲笑我们的人：假如说我们是在那里幻想，那么你们究竟什么时候才能不靠基督，只凭自己的智慧盖起大厦，建立起合理的生活来呢？

## 《动物农场》

雪球同志在哪儿？托洛茨基在哪儿？

必须按正规方式称“我们的领袖拿破仑同志”，而猪们则喜欢为他发明创造诸如所有动物之父、人见愁、羊圈守护神、小鸭之友之类的头衔。吱嘎在讲演时总是泪流满面地谈到拿破仑的智慧何等超群，他的心地多么善良，他对任何地方的所有动物怀着深深的爱，甚至而且尤其深爱其他农场至今仍生活在愚昧和被奴役状态的不幸动物。

故事没有给一个开放的结尾，拿破仑带着他的五条猎犬和代表着资本家的人类达成了友好共识，以同样形式压榨动物农场的动物，不对，动物农场的名字已经被改回了庄园农场。

一个遗憾是，在拿破仑夺权后，故事情节缺少转折点，导致故事一边倒的倒向反乌托邦社会，读完让读者感到压抑和失落。添加动物做出反抗但最后被压迫的情节是否会更好？

# 正在阅读

## 《荒原狼》 - 黑塞

有人评价这本书是德国的《尤利西斯》，当初第一次听尤利西斯的介绍时被其意识流吸引，据说其每个章节采用了不同的写作手法，各种引经据典，天花乱坠。还有一点是有人在《地下室手记》的讲解视频里提到了这本书，于是乎勾起了我阅读兴趣，结果出乎我的意料。

### 乏味空洞的前言

刚开始阅读前言，感觉回到了《刀锋》，那本所谓‘二流作家’毛姆撰写的小说，通篇以第三人称视角描写主人公，这种描写方式淡化了主人公的心理活动，弱化了故事情节，降低了故事的真实感。第三称叙事大大削弱了整本书的可读性。前言部分内容以我的视角描写主人公，描绘他多么与众不同，但是显得很空洞，不真实。

但在我看完整本书后，发现前言的作用在于引出主角的笔记，丰富主角的性格，与后文的手记内容做映衬对比。

### 魔幻现实主义与意识流

后面开始讲一个愤世弃俗的主人公的经历，充满了意识流。后面的魔力剧场直接看懵了，有一种万花筒的感觉。只能说是一种奇妙的体验。记忆最深的是回忆与初恋女友的经历，在大树下，山坡上，绿草地里，真挚的告白，心理描写动作描写刻画十分到位，产生了共鸣。

我站在我居住的那座小镇上方的山嘴上。风将春天与紫罗兰的气息送进了我的长发里面。我在山下的小镇上看到了波光粼粼的小河，还看到了我家的窗户，我看到的、听到的、闻到的一切深深地触动了我。那种感觉就像新鲜的、还站不稳的生……

最后的解决也是呼应前面的伏笔，以一场意外的但又注定的方式，主角杀害了他喜欢的人，感觉比较意识流，故事情节一下子被串起，伏笔终于被揭开，有始有终。

### 一个比较好奇的点

主人公之前是一名五十多岁的落魄流浪人，失去了中产阶级生活，失去了家庭和工作，留有一点财产，喜欢的人与他互不往来。每天独来独往，痛恨现代的声色犬马，靡靡之音，延误现代人尽情纵欲的生活，一个重要的反复被提及的点是：他喜欢贝多芬，而现代人热爱爵士乐——这是主人公所深深痛恨的，因为他认为这根本就不算音乐。甚至不忍让别人用唱片机播放贝多芬的音乐，认为这是一种侮辱。

他过着离群索居的生活，愤世弃俗来形容他好不为过，感觉对事事都很认真，没有一点幽默感。但是这样的他偶然喜欢上了一个酒吧女，并且她教会了他跳舞（舞蹈应该算一个重要的线索），并且她还给他介绍了性伴侣。（后面也提及到一个乐手提出要三人行……）

为我们每人倒了一杯东西，这东西神秘又奇妙。然后，在我心情好起来以后，他两眼放着光，提议我们三

个人好好地搞一场性交。我断然拒绝了。这种污秽的勾当在我是不可想象的。然而，我还是偷偷地瞥了一眼玛丽亚，竟发现她的眼睛也闪着光，还说不搞这事让她觉得好遗憾。巴伯罗看我没同意显得很失望，却没受到伤害。

性伴侣这里是一个主人公心态的转折点，他从性伴侣上获得了生命的活力，这一段描述的十分夸张：他们昼夜不停，各种姿势，主人公感觉回到了年轻的时候，重新对生活充满了激情，精神焕发，仿佛变了一个人。

一个比较好奇的点是，真的有这么大的魔力吗？

### 反思现代化

这个时代的目标没有一个是我喜欢的，这个时代的快乐没有一种是我看得上的，可为何我还是没能变成一匹孤独的狼，一位粗野的隐士？无论在剧场，还是在影院，我都坐不长。报纸我几乎看不下去，现代的书籍几乎没读过。我不懂到底是因为什么样的快乐和欢愉驱使着人们赶往人满为患的火车站、旅馆，驱使着人们进入充斥着生硬、令人窒息的音乐的人满为患的餐馆，驱使着人们赶往酒吧、各式各样的娱乐场所、世界博览会以及休闲购物商业街。别的人，数以万计的人，都争着抢着去追求这样的快乐，但这样的快乐我不懂，也无法与人分享，尽管我能轻易地得到它。相反，我的生活中快乐的时候并不多，在我看来，生活、快乐、狂喜、兴……
