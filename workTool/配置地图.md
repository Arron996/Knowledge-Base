# 配置地图

各配置在磁盘上的位置与职责。路径均为本机绝对路径。

## Cursor 核心

| 路径 | 职责 |
|------|------|
| `~/IdeaProjects/.cursor/rules/*.mdc` | 工作区 Rules（业务 + 工作流通用） |
| `~/IdeaProjects/.cursor/skills/*/SKILL.md` | 工作区 Skills（FAT/排障/发布等） |
| `~/IdeaProjects/.cursor/scripts/` | F2B/MR/钩子相关 shell |
| `~/IdeaProjects/.mcp.json` | 工作区 MCP 定义（子集，AGENTS 对齐用） |
| `~/IdeaProjects/AGENTS.md` | Codex ↔ Cursor 对齐总述 |
| `~/.cursor/mcp.json` | **用户级 MCP（最全）** |
| `~/.cursor/hooks.json` | Agent Hooks（Shell→rtk、git-ai checkpoint） |
| `~/.cursor/skills-cursor/` | Cursor 官方/内置 Skills |
| `~/.cursor/skills/` | 用户级业务 Skills（与 `~/.claude/skills/` 镜像） |
| `~/.cursor/extensions/` | VS Code 兼容扩展 |
| `~/.cursor/plugins/cache/cursor-public/superpowers/` | Superpowers 插件 Skills |
| `~/Library/Application Support/Cursor/User/settings.json` | 编辑器用户设置 |
| Cursor Settings → Rules | User Rules（对话级持久约定，UI 配置） |

## 事实源优先级（改配置时）

1. **MCP 列表以 `~/.cursor/mcp.json` 为准**（比 IdeaProjects `.mcp.json` 多 captain / arex / qmq / gds_soa / codegraph）。
2. **Rules / Skills 以 `IdeaProjects/.cursor/` 为准**（AGENTS.md 也指向这里）。
3. **日报文风以 `Templates/daily-style-guide.md` 为准**。
4. **F2B 执行细节以 `git-f2b-workflow.mdc` 为准**；Plan 时先读 `git-f2b-branch-policy.mdc`。
