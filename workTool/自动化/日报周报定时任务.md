## 早间待办（09:00）

Instructions：`Templates/automation-morning-instructions.md`

1. `workday.py --prev` 取**上一工作日**（跨周末/节假日）
2. `roll-morning-todos.py`：昨日「明日待办」→ 今日「今日计划」；待排期去重合并

4. git commit + push

纯脚本，**不调 SDK**。

## Daily 结构约定（Agent「记待办」也遵守）

| 区块 | 含义 |
|------|------|
| `## 📋 今日计划` | 9:00 滚入；🔴🟡🟢 |
| `## 📋 明日待办` | 承诺明日做 |
| `## 🗂️ 待排期` | 未排期；🔵 |
| `## 📊 技改进展` | 业务技改 + 进度符号 |
| `## 📎 参考` | 分支/MR 等工程明细（折叠） |

进度符号见 Dashboard / style-guide（📐✏️🔍…✅）。

对话口令示例：「更新日报，待排期：🔵 …」——走 `kb-daily-todo` Rule，**不进飞书技改 Wiki**。
