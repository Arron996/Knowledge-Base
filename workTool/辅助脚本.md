# 辅助脚本

## IdeaProjects `.cursor/scripts/`

| 脚本 | 作用 |
|------|------|
| `mr-auto-merge-approved.sh` | 已批准 MR 自动合入（被 `sync-my-mrs-scheduled.zsh` 调用） |
| `install-pre-commit-hooks.sh` | 安装 pre-commit |
| `pre-commit-format.sh` | 提交前格式化 |
| `fat-qmq-send.sh` | FAT 发 Q 辅助 |

## `~/bin/`（定时入口）

| 脚本 | 作用 |
|------|------|
| `kb-morning-launch.sh` | 早间 launchd → 知识库 morning |
| `kb-evening-launch.sh` | 晚间 launchd → 知识库 evening |
| `sync-my-mrs.zsh` | 同步「我的 MR」状态 |
| `sync-my-mrs-scheduled.zsh` | 定时 wrapper + 自动合入 |

## Pre-commit / git-ai

- `git.path` 指向 `~/.git-ai/bin/git`
- Hooks 在工具调用前后做 `git-ai checkpoint`
- Shell 经 `rtk hook cursor`（见 [[Cursor/插件Hooks扩展]]）
